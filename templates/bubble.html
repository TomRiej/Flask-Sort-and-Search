Bubble Sort

Bubble sort is a sorting algorithm which looks through every pair of elements in a list. If its sorted in ascending order, if element 1 is greater than element 2, they swap so that the pair is in order. This is repeated until no swaps are made.


O(n^2) big-Omega(n)
